# MTG Pack Opening Simulator

## Overview
Simulates opening Magic: The Gathering packs. It randomly generates a pack based on card rarity and type distribution and dynamically fetches card data via the Scryfall API,
including names, images, and prices

## Built With
Java  
JavaFX  
Maven

## How to run
1.) Clone repository  
2.) Navigate to project directory and compile  
3.) Run the program
